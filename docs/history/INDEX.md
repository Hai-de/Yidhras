# 历史归档 / History Archive

本目录存放面向读者的稳定历史背景文档：

- 已完成的重大迁移记录（如 Rust 迁移各阶段的边界变化）
- 已定稿的设计决策历史（重大架构转向的原因与过程）
- 系统演进上下文（帮助新读者理解"为什么系统长这样"）

## 与 `.limcode/archive/` 的区别

| 维度 | `docs/history/` | `.limcode/archive/` |
|------|-----------------|---------------------|
| 受众 | 所有读者 | 项目内部维护者 |
| 内容 | 已成事实的历史背景与决策理由 | 已完成的过程资产（设计草案、计划、评审） |
| 性质 | 面向读者的叙事性历史 | 面向执行的工程过程记录 |
| 稳定性 | 稳定，不随开发迭代频繁变更 | 随过程推进持续归档 |

当前尚无已归档的历史文档。重大迁移或架构转向完成后，在此添加对应条目并更新本索引。
