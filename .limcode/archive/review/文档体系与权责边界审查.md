- 严重级别: 高

- 跟踪状态: 开放

- 说明:

  README 自称只保留概览、启动、常用命令与导航，但实际又包含插件治理 CLI 细节、定向测试命令、路由入口和大段实现概览，已经从“入口页”膨胀为“操作手册 + 状态快照 + 子系统索引”。这会让 README 与 AGENTS、docs/INDEX、子系统文档同时争夺入口地位。
