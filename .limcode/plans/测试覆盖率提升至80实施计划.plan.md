## 3. 技术实施细节

### 3.1 服务器端测试策略
- **单元测试**: 70%覆盖率目标，使用vitest
- **集成测试**: 60%覆盖率目标，使用测试数据库
- **E2E测试**: 30%覆盖率目标，使用Playwright

### 3.2 关键测试模式
- **mock-context 模式**（最高效）：`createMockAppContext()` + `ctx.prisma.xxx.mockResolvedValue()`
- **TestApp 路由模式**：`createTestApp(ctx, { operator })` + `app.get('/api/...')`
- **纯函数测试**：直接 import + assert

### 3.3 测试质量保证
- **代码审查**: 每个测试文件必须经过审查
- **测试覆盖率门禁**: PR合并前检查覆盖率
- **测试维护**: 定期清理过时测试
