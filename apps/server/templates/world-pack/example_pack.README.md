# Example World Pack

> 一个用于演示通用 world-pack 合约的中性参考包。

## 概览

- Pack ID：`world-example-pack`
- Version：`1.0.0`
- 当前状态：bundled example template
- 运行时主配置：`config.yaml`

## 定位

这个包的作用是：

- 为项目提供中性的 bundled example template
- 演示最小可运行的 world-pack contract
- 作为脚手架与默认运行目录的参考样本

它不是宿主核心默认世界，也不承载任何特定题材语义。

## 使用方式

将其作为示例 world-pack 放入：

```text
data/world_packs/example_pack/
```

## 边界

- 宿主核心定义 world-pack 框架
- 示例包只演示一种最小内容组织方式
- 具体题材语义应由各自的 world-pack 自行承载
